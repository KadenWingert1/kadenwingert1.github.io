Pokemon Game

Description

The repository contains code for a rogue-like, text-based Pokemon game originally implemented in C. The code has undergone changes, including a port to C++. The game features a semi-randomly generated world, various trainers, battles, Pokemon Centers, Pokemarts, and more. Data is loaded from multiple csv files, and the game involves interacting with Pokemon data types, trainers, and battles. A makefile was used to compile the source code and dependenceies. The controls include movement, opening interfaces, resting, and various actions during battles.
